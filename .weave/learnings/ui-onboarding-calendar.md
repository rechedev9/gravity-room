# Learnings: ui-onboarding-calendar

## Task 16: Scheduling real opcional con metadata, sin migración

- **Discrepancy**: El plan condiciona esta tarea a que el usuario confirme que quiere calendario real planificado, pero no existe confirmación explícita en la sesión actual.
- **Resolution**: Se marcó como bloqueada/omitida para evitar introducir scheduling futuro, metadata nueva o UX de fechas reales sin aprobación de producto.
- **Suggestion**: Separar el scheduling real planificado en un plan posterior con decisión explícita sobre fecha de inicio, días preferidos y edición de metadata.
